Markov Literature Test

This is a webapp that uses Markov chains to create sentences meant to mimic James Joyce's novel Ulysses. The user is shown a Markov chain and a real random passage from the book, and then they have to guess which one is the real passage.

Languages: Python, HTML, CSS

Technologies/Frameworks:Flask, Semantic-UI

The chains are generated with help from the Markovify api that can be found here:

https://github.com/jsvine/markovify
